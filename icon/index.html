    Icon

    Semantic vector graphics.

    Examples

    Basic

    Svant offers a list of icons that you can import from svant/icons, there are components for icons with different themes, you can use spin to animate the icon, and rotate to rotate it.

    View Code
    <HomeOutlined />
<SettingFilled />
<SmileOutlined />
<SyncOutlined spin />
<SmileOutlined rotate="{180}" />
<LoadingOutlined spin />

<script>
  import {
    HomeOutlined,
    LoadingOutlined,
    SettingFilled,
    SmileOutlined,
    SyncOutlined
  } from "svant/icons";
</script>

    Two-tone icons

    You can set twoToneColor prop to specific primary color for two-tone icons.

    View Code
    <SmileTwoTone />
<HeartTwoTone twoToneColor="#eb2f96" />
<CheckCircleTwoTone twoToneColor="#52c41a" />

<script>
  import { CheckCircleTwoTone, HeartTwoTone, SmileTwoTone } from "svant/icons";
</script>

    Common Attributes

    PropertyDescriptionTypeDefault
    classClass of IconString | Object-
    styleStyle properties of icon, like fontSize and colorString | Object-
    spinRotate icon with animationBooleanfalse
    rotateRotate by n degrees (not working in IE9)Number-
    twoToneColorOnly supports the two-tone icon. Specify the primary color.String (hex color)-

    If you are using the Icon component these are additional attributes besides the above
